Primeros pasos

Usuario

$ git config --global user.name "Maxx"
$ git config --global user.email maximiliano@fizzmod.com

Extra pero útil

$ git config --global credential.helper cache
$ git config --list

Empezar con un repositorio

Crear uno

$ git init

Obtener uno existente

$ git clone https://github.com/{user}/{repo}.git {alias}(opcional)

Comandos básicos

Actualizar repositorio local con el del server

$ git pull

Agregar al área de preparación (Staging Area)

$ git add .

Confirmar lo que se encuentra en área de preparación

$ git commit -a(opcional) -m "Mensaje del commit" | --amend

Actualizar el server con el contenido local

$ git push

Actualizar repositorio local

$ git fetch

Estado actual de tu area de trabajo con respecto al server

$ git status (-s | --short)

Ver lineas modificadas o eliminadas

$ git diff (--staged/cached | --stat)

Ver historial de confirmaciones

$ git log (-p | -2 | --stat | --graph | --oneline)

Volver archivos al estado inicial

$ git checkout .

Deshacer archivos preparados para confirmar

$ git reset HEAD .
